# 🚀 Astro + ReScript + Tailwind CSS Portfolio

A modern, type-safe web development stack combining **Astro's island architecture**, **ReScript's functional programming**, and **Tailwind CSS v4** for building fast, interactive portfolios.

## ✨ Features

- 🏝️ **Astro v6** - Fast static site generation with islands architecture
- 🔒 **ReScript** - 100% type-safe functional language compiling to JavaScript/ESM
- ⚛️ **React 19** - Interactive components with selective hydration
- 🎨 **Tailwind CSS v4** - Utility-first CSS with automatic class detection from `.mjs` files
- 🌙 **Dark Mode** - Theme toggle with class-based dark mode support
- 💼 **Portfolio Components** - Reusable Button, Card, Counter components
- 📦 **ES Modules** - ReScript compiles to `.mjs` for modern module support
- 🔄 **Hot Reload** - Concurrent watch mode for both ReScript and Astro

## 📋 Prerequisites

Before you begin, ensure you have installed:

- **Node.js** >= 22.12.0
- **npm** or **pnpm** or **yarn**
- **Git** (optional, for cloning)

## 🏗️ Project Setup from Scratch

### Step 1: Clone or Create New Project

```bash
# Clone this repository
git clone <your-repo-url>
cd abinash.nagaraj-portfolio

# OR create from scratch
npm create astro@latest my-portfolio -- --template minimal
cd my-portfolio
```

### Step 2: Install Dependencies

```bash
npm install
```

This installs:

- `astro` - Static site framework
- `@astrojs/react` - React integration for Astro
- `@tailwindcss/vite` - Tailwind CSS v4 Vite plugin
- `rescript` - ReScript compiler
- `@rescript/react` - ReScript React bindings
- `concurrently` - Run multiple commands in parallel

### Step 3: Configure ReScript

Create `bsconfig.json`:

```json
{
  "name": "abinash.nagaraj-portfolio",
  "sources": [
    {
      "dir": "src/components/rescript",
      "subdirs": true
    }
  ],
  "package-specs": [
    {
      "module": "esmodule",
      "in-source": true
    }
  ],
  "suffix": ".mjs",
  "jsx": {
    "version": 4,
    "mode": "automatic"
  },
  "dependencies": ["@rescript/react"]
}
```

**Key settings:**

- `"module": "esmodule"` - Output ES modules
- `"in-source": true` - Generate `.mjs` files alongside `.res` files
- `"suffix": ".mjs"` - Use `.mjs` extension instead of `.js`
- `"jsx": { "version": 4, "mode": "automatic" }` - React 19 support

### Step 4: Configure Astro

Update `astro.config.mjs`:

```javascript
import { defineConfig } from "astro/config";
import react from "@astrojs/react";
import tailwindcss from "@tailwindcss/vite";

export default defineConfig({
  integrations: [react()],
  vite: {
    plugins: [tailwindcss()],
    build: {
      rollupOptions: {
        output: {
          assetFileNames: (assetInfo) => {
            if (assetInfo.name?.endsWith(".css")) {
              return "index.css";
            }
            return "[name].[hash][extname]";
          },
        },
      },
    },
  },
});
```

### Step 5: Configure Tailwind CSS

Create `src/styles/global.css`:

```css
@import "tailwindcss";

/* Tell Tailwind to scan .mjs files generated by ReScript */
@source "../**/*.mjs";
@source "../**/*.astro";

/* Configure dark mode variant */
@variant dark (.dark &);
```

**Important:** Tailwind v4 scans `.mjs` files to detect classes used in ReScript components.

### Step 6: Update package.json Scripts

```json
{
  "scripts": {
    "dev": "concurrently \"npm run res:dev\" \"astro dev\"",
    "build": "npm run res:build && astro build",
    "preview": "astro preview",
    "res:build": "rescript",
    "res:dev": "rescript watch",
    "res:clean": "rescript clean"
  }
}
```

### Step 7: Configure .gitignore

Add ReScript build artifacts:

```gitignore
# ReScript build artifacts
lib/
src/**/*.mjs
```

## 🛠️ Development Workflow

### Start Development Server

```bash
npm run dev
```

This runs **two processes concurrently**:

1. **ReScript Watch** - Compiles `.res` → `.mjs` on file changes
2. **Astro Dev Server** - Serves at `http://localhost:4321` with HMR

**Output:**

```
[0] rescript watch
[0] Compiled 6 modules
[1] astro dev
[1] Local: http://localhost:4321/
```

### Create a New ReScript Component

1. **Create the component file:**

```rescript
// src/components/rescript/Greeting.res
@react.component
let make = (~name: string) => {
  <div className="p-6 bg-gradient-to-r from-blue-500 to-purple-600 text-white rounded-lg shadow-lg">
    <h2 className="text-2xl font-bold">
      {React.string(`Hello, ${name}!`)}
    </h2>
  </div>
}
```

2. **ReScript auto-compiles** to `Greeting.mjs` (watch mode)

3. **Import in Astro page:**

```astro
---
import { make as Greeting } from '../components/rescript/Greeting.mjs';
---

<html>
  <body>
    <Greeting name="World" client:load />
  </body>
</html>
```

4. **Tailwind detects** classes from `Greeting.mjs` and includes them in `index.css`

### Build for Production

```bash
npm run build
```

**Build steps:**

1. `rescript` - Compiles all `.res` files to `.mjs`
2. `astro build` - Bundles and optimizes
3. Output: `dist/index.html` and `dist/index.css`

**Build output:**

```
dist/
├── index.html          # Minified HTML
├── index.css           # Optimized Tailwind CSS (only used classes)
├── _astro/
│   ├── client.*.js     # React hydration runtime
│   └── HomePage.*.js   # Bundled component code
├── favicon.svg
└── favicon.ico
```

### Preview Production Build

```bash
npm run preview
```

Serves the `dist/` folder at `http://localhost:4321`

## 🎨 Key Concepts

### ReScript to React Flow

```
Button.res  →  [ReScript Compiler]  →  Button.mjs  →  [Astro/Vite]  →  Bundled JS
```

### Tailwind CSS Detection Flow

```
Button.res (className="bg-blue-500")
    ↓
Button.mjs (className: "bg-blue-500")
    ↓
Tailwind scans .mjs (via @source "../**/*.mjs")
    ↓
index.css includes .bg-blue-500
```

### Dark Mode Implementation

**ThemeContext.res:**

```rescript
let (theme, setTheme) = React.useState(() => Dark)

let className = switch theme {
| Dark => "dark"    // Adds 'dark' class to wrapper div
| Light => ""
}

<Provider value={{theme, toggleTheme}}>
  <div className> {children} </div>
</Provider>
```

**Usage in components:**

```rescript
<div className="bg-white dark:bg-gray-800 text-gray-900 dark:text-white">
  {React.string("Adapts to theme!")}
</div>
```

## 🔧 Configuration Details

### Astro Client Directives

Use `client:load` for interactive ReScript components:

```astro
<HomePage client:load />     <!-- Hydrates on page load -->
<Counter client:visible />   <!-- Hydrates when visible -->
<Card client:idle />         <!-- Hydrates when browser idle -->
```

### ReScript Module Exports

ReScript components export a `make` function:

```javascript
// Compiled output in .mjs
export { make };
```

Import in Astro:

```javascript
import { make as ComponentName } from "./Component.mjs";
```

### Tailwind Class Detection

Tailwind v4 uses `@source` to watch files:

```css
@source "../**/*.mjs"; /* Scans ReScript output */
@source "../**/*.astro"; /* Scans Astro files */
```

## 🐛 Troubleshooting

### ReScript compilation errors

```bash
# Clean and rebuild
npm run res:clean
npm run res:build
```

### Tailwind classes not applying

1. Check `src/styles/global.css` has `@source "../**/*.mjs"`
2. Verify `.mjs` files are generated in `src/components/rescript/`
3. Hard refresh browser: `Ctrl + Shift + R`

### Port already in use

```bash
# Kill existing processes
Stop-Process -Name "node" -Force  # Windows
kill -9 $(lsof -ti:4321)          # macOS/Linux

# Or change port in astro.config.mjs
server: { port: 3000 }
```

### Module not found errors

Ensure correct import path:

```javascript
// ✅ Correct
import { make as HomePage } from "../components/rescript/HomePage.mjs";

// ❌ Wrong - default import doesn't exist
import HomePage from "../components/rescript/HomePage.mjs";
```

## 🧩 Component Hydration

Use Astro's client directives to control when components load:

- `client:load` - Hydrate immediately
- `client:idle` - Hydrate when browser is idle
- `client:visible` - Hydrate when visible in viewport
- `client:only="react"` - Only render on client

Example:

```astro
<Counter client:idle />
```
